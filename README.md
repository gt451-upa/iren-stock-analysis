# iren-stock-analysis
# IREN Stock Analysis 

## Overview
This project analyzes the stock performance of **IREN (Iris Energy)**, a Bitcoin mining and AI data center company listed on NASDAQ. The analysis includes price history, daily returns, volatility metrics, and correlation with Bitcoin.

## Key Findings
- Annualized return, volatility and Sharpe Ratio of IREN since 2022
- Strong correlation between IREN and Bitcoin price movements
- Visual comparison of IREN vs Bitcoin returns

## Tools & Libraries
- Python
- yfinance (data collection)
- pandas & numpy (data processing)
- matplotlib (visualization)

## Contents
- `iren_analysis.ipynb` – Main analysis notebook

## How to Run
1. Clone the repository
2. Install dependencies: `pip install yfinance pandas numpy matplotlib`
3. Open `iren_analysis.ipynb` in Jupyter and run all cells
